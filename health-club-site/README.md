# Bloom Health Club – Community Wellness Hub

A functional React prototype for a health club website that enables community organizers to collaborate and plan wellness events together.

## Tech Stack

- **React 19** with Vite
- **Tailwind CSS v4** with a custom lavender & white theme
- **Inter** font (Google Fonts)

## Features

- 🌿 **Hero Section** – Eye-catching landing with stats (members, events, organizers, communities)
- 📅 **Events Page** – Browse & filter upcoming health events by category; register/unregister with spot tracking
- 🤝 **Organizers Page** – Meet the community leaders behind each event, with bios and specialty tags
- 📸 **Gallery** – Visual highlight reel of past community events
- ✉️ **Contact / Join** – Contact info cards + functional inquiry form with submission confirmation

## Color Theme

Built with a custom lavender palette (`lavender-50` → `lavender-900`) layered on white backgrounds — soothing, accessible, and community-friendly.

## Getting Started

```bash
cd health-club-site
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

```bash
npm run build
npm run preview
```
